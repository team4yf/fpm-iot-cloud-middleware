# Protocol 00

以 00 开头的协议标准。用于适应 NBIOT 应用开发的项目中。

## Data Packet Define

```javascript
// The hex data
/*
----------------------------------------------------------------------------------------------------------------
| - VID  | - UID |  ProjectID  |          NBID           |      SN      | FN | EXTRA |      DATA...    |  CRC  |
--------------------------------------------------------------------------------------------------------------
| - 1B - | - 3B  | ---- 4B --- | --------  8B ---------- | ---- 4B ---- | 1B | - 2B -| ----- ?B ------ | - 2B  |
--------------------------------------------------------------------------------------------------------------
                                                         |  ----------------   Origin DATA   ----------------  |
----------------------------------------------------------------------------------------------------------------

*/
```

Fields

- VID

  The Protocol ID. Defined `0`.
- UID

  The User ID.
- ProjectID

  The Project ID.
- NBID

  The Device's NBID.
- SN

  The Device's SN Code.
- FN

  The Function Code.
- EXTRA

  The Extra Code.
- DATA

  The Main Data.
- CRC

  The CRC16 Compare Code.
